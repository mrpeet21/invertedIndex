# 3 Модуль: Inverted Index + Elias Gamma/Delta

## Что делает код

1. Загружает исходный файл `pages.csv.gz`.
2. Берёт колонки:
   - `id`
   - `final_url`
   - `title`
   - `text_content`
3. Выполняет предобработку текста:
   - нижний регистр;
   - токенизация;
   - лемматизация русских и английских слов.
4. Строит инвертированный индекс.
   Сжимает posting lists через:
   - Gap + Elias Gamma;
   - Gap + Elias Delta.
5. Сравнивает:
   - размер обычного индекса;
   - размер Gamma-сжатого индекса;
   - размер Delta-сжатого индекса;
   - скорость boolean AND поиска по обычному и сжатым индексам.
6. Дополнительно (фразовый поиск):
   - строит позиционный индекс
   - определяет время фразового поиска


## Как запустить

### 0. Скачать данные

Спарсенные данные статей Хабра можно скачать по ссылке: https://cloud.mail.ru/public/M9pe/2vX2eSJMg 
Данный файл pages.csv.gz нужно поместить в каталог data/

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Проверить CSV

```bash
python check_csv.py
```

### 3. Установить сертификаты (опционально)

```bash
//Пример для MacOS
/Applications/Python\ 3.12/Install\ Certificates.command
```

### 4. Запустить основной код

```bash
python main.py
```

## Быстрый тест на части документов

В `main.py` можно временно поставить:

```python
MAX_DOCS = 1000
```

Для полного запуска по всем документам:

```python
MAX_DOCS = None
```

## Как поменять поисковый запрос

В `main.py` измени строку:

```python
QUERY_TEXT = ""
```

Например:

```python
QUERY_TEXT = "Машинное обучение"
```

или:

```python
QUERY_TEXT = "accuracy"
```
